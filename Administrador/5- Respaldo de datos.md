
Para mantener segura la información  que es ingresada a las aplicaciones el sistema crea respaldos de los datos ingresados en sheets todos los días en la carpeta llamada "Respaldos_DB_Ds49 o Respaldos_DB_Ds19" dependiendo el proyecto.

![[Pasted image 20260615154139.png|341]]

En caso de pérdida de información en la base de datos principal del proyecto, el administrador podrá recuperar la información mediante estos respaldos y/o mediante el registro de actividades de sheets.
