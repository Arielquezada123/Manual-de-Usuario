
Para dar a los usuarios nuevos acceso a los informes generados nos dirigiremos a Drive a la carpeta donde se almacenan nuestros archivos con la cuenta en donde se aloja el sistema, la carpeta raíz es "Sistema de seguimiento DS19/DS49".

![[Pasted image 20260611164422.png|513]]

Dentro de ella veremos las dos carpetas que separan los formatos de informe y base de datos de cada aplicación junto con sus respectivas imágenes y/o firmas, entraremos en una carpeta primero teniendo en cuenta que repetiremos el proceso en las dos

![[Pasted image 20260611170201.png|515]]

Presionaremos la carpeta "Informes" y el botón de "Compartir"

![[Pasted image 20260611170420.png|515]]

Y ya podríamos agregar el correo electrónico de los nuevos usuarios de el sistema 

![[Pasted image 20260611170741.png|451]]

Repetiríamos el mismo proceso para los dos proyectos DS19 y DS49 dando solo permiso a la carpeta "Informes", las otras carpetas y archivos como las imágenes, firmas, base de datos y el formato de informe quedan solo disponibles para el usuario administrador
